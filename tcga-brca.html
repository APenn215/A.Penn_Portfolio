Overview

This project explores molecular differences between Basal and Luminal A breast cancer subtypes using transcriptomic analysis and predictive modeling. It serves as the centerpiece of the portfolio because it aligns directly with bioinformatics, cancer genomics, and precision medicine research.

Research / Analysis Objective

Evaluate whether gene expression patterns can distinguish breast cancer molecular subtypes and identify biological pathways associated with subtype differences.

Methods

  • Preprocessed gene expression data and selected informative features
  • Applied PCA to explore transcriptomic separation between subtypes
  • Performed differential expression analysis to identify subtype-associated genes
  • Used pathway enrichment concepts to interpret biological mechanisms
  • Built classification models such as Logistic Regression and Random Forest

Key Results

  • Basal and Luminal A subtypes showed distinguishable expression patterns
  • Basal subtype patterns were associated with cell cycle, mitosis, and DNA replication pathways
  • Luminal A patterns were associated with extracellular matrix, metabolic, and oxidative processes
  • Machine learning models reached approximately 89% classification accuracy
