Professor Evaluation Regression Modeling

Statistics • Regression • Bias & Evaluation Analysis

Download NotebookContact Me

Overview

This project uses multiple linear regression to examine factors associated with university professor evaluation scores. It highlights statistical modeling, diagnostics, and responsible interpretation of predictors.

Research / Analysis Objective

Identify variables associated with professor evaluation scores and evaluate whether appearance-related ratings remain predictive after accounting for other variables.

Methods

  • Explored the distribution of student evaluation scores
  • Visualized relationships among predictors including beauty ratings and evaluation scores
  • Built simple and multiple linear regression models using statsmodels
  • Assessed statistical significance, coefficients, and practical significance
  • Used backward selection with p-values to identify a final model
  • Evaluated residual plots, histograms, and Q-Q plots for model diagnostics

Key Results

  • Beauty rating was statistically significant but explained a limited amount of variation alone
  • The final model included ethnicity, gender, language, age, class evaluation percentage, class credits, beauty rating, and picture color
  • Diagnostic plots suggested the final model was reasonably acceptable but residuals showed some left skew
  • The project emphasized limitations in generalizing results beyond the original university dataset
