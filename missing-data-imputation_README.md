# Missing Data Analysis and Imputation

## Overview
This project evaluates how missing values affect biomedical datasets and compares multiple imputation methods that are commonly used before statistical modeling or machine learning.

## Objective
Assess missingness patterns and compare mean, KNN, and regression-based imputation methods to prepare a dataset for downstream analysis.

## Methods
- Loaded complete and missing-value versions of an insertion depth dataset
- Visualized missing data using dendrograms, matrix plots, bar charts, and heatmaps
- Applied mean imputation using SimpleImputer
- Applied KNN imputation and regression-based imputation workflows
- Compared original and imputed distributions with histograms and boxplots

## Key Results
- Created a structured workflow for identifying and visualizing missingness
- Compared how different imputation strategies changed feature distributions
- Showed that imputation method selection can influence data quality and analysis readiness
- Produced visual outputs that communicate preprocessing decisions
