# Iris Flower Species Classification

## Overview
This project applies supervised machine learning to classify Iris flower species using sepal and petal measurements. It demonstrates a complete beginner-to-intermediate classification workflow.

## Objective
Build and compare classification models that predict Iris species from four measured features.

## Methods
- Loaded the Iris dataset from a public GitHub source
- Separated features and labels
- Applied feature normalization
- Split the data into training and testing sets using stratification
- Trained Logistic Regression, Decision Tree, Random Forest, and Neural Network classifiers
- Compared model performance with test accuracy, classification reports, and 5-fold cross-validation

## Key Results
- All models performed strongly on the Iris classification task
- Random Forest showed the most stable performance across validation tests
- Decision Tree was interpretable and effective but can be more prone to overfitting
- Neural Network performed well but was less necessary for a simple dataset
