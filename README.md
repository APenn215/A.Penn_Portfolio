# A'Brea Penn Bioinformatics Portfolio Website

This repository contains a GitHub Pages-ready portfolio website for bioinformatics, healthcare data science, and research applications.

## Featured Projects

- Breast Cancer Transcriptomic Analysis
- Missing Data Analysis and Imputation
- Iris Flower Species Classification
- Professor Evaluation Regression Modeling
- Sudden Cardiac Death Prediction in Heart Failure Patients

## How to Publish with GitHub Pages

1. Create a new GitHub repository named `bioinformatics-portfolio`.
2. Upload all files from this folder.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save. Your website link will appear after GitHub finishes publishing.
